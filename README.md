# AI Chatbot SaaS

A full-stack AI chatbot application built using React and FastAPI.

## Features
- Real-time chat interface
- AI-powered responses using LLM API
- Context-based conversation
- REST API backend

## Tech Stack
- React.js
- FastAPI
- Python
- OpenAI API

## How to Run

### Backend
cd backend
uvicorn app.main:app --reload

### Frontend
cd frontend
npm start
